## Установка

1. Клонирование репозитория

`git clone https://github.com/1habibi/Boxly2.0Mobile.git`

2. Переход в директорию

`cd Boxly2.0Mobile`

3. Установка зависимостей

`npm install`

4. Настройка .env файла

5. Запуск скрипта для запуска сервера

`npm expo start -c`
